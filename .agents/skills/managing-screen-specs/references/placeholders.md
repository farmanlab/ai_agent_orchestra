# プレースホルダー定義

画面仕様書テンプレートで使用するプレースホルダーの一覧と置換ルール。

## 概要セクション

| プレースホルダー | 説明 | 例 |
|-----------------|------|-----|
| `{{SCREEN_NAME}}` | 画面名 | TOP画面, 検索結果画面 |
| `{{SCREEN_ID}}` | 画面識別子 | top, search-results |
| `{{FIGMA_URL}}` | Figma URL | https://figma.com/design/xxx?node-id=123 |
| `{{HTML_FILE}}` | HTMLファイル名 | top.html |
| `{{ROOT_NODE_ID}}` | FigmaノードID | 2350:2662 |
| `{{DATE}}` | 日付 | 2025-01-14 |
| `{{DESCRIPTION}}` | 画面説明 | AskAI機能のエントリーポイント画面 |

## 構造・スタイルセクション

| プレースホルダー | 説明 | 例 |
|-----------------|------|-----|
| `{{HTML_STRUCTURE}}` | HTML構造 | ルート要素とdata-figma属性 |

## コンテンツ分析セクション

| プレースホルダー | 説明 | 例 |
|-----------------|------|-----|
| `{{CONTENT_NAME}}` | コンテンツ名 | ユーザー名, 投稿日時 |
| `{{DATA_ATTRIBUTE}}` | data属性 | data-content="user-name" |
| `{{API_SOURCE}}` | API出所 | GET /api/users/:id |

## UI状態セクション

| プレースホルダー | 説明 | 例 |
|-----------------|------|-----|
| `{{ELEMENT_NAME}}` | 要素名 | 送信ボタン, 入力フィールド |
| `{{ELEMENT_STATE}}` | 状態 | 有効, 無効, ホバー |
| `{{ELEMENT_NOTES}}` | 備考 | #0070e0背景 |

### ボタン状態プレースホルダー

| プレースホルダー | 説明 |
|-----------------|------|
| `{{BUTTON_DEFAULT_BG}}` | デフォルト背景色 |
| `{{BUTTON_DEFAULT_TEXT}}` | デフォルトテキスト色 |
| `{{BUTTON_HOVER_BG}}` | ホバー背景色 |
| `{{BUTTON_DISABLED_BG}}` | 無効時背景色 |
| `{{BUTTON_DISABLED_TEXT}}` | 無効時テキスト色 |

## アクセシビリティセクション

| プレースホルダー | 説明 | 例 |
|-----------------|------|-----|
| `{{ELEMENT}}` | 対象要素 | 送信ボタン |

## 画面フローセクション

| プレースホルダー | 説明 | 例 |
|-----------------|------|-----|
| `{{CURRENT_SCREEN}}` | 現在画面 | TOP |
| `{{NEXT_SCREEN}}` | 遷移先画面 | 詳細 |
| `{{TRIGGER_ACTION}}` | 遷移トリガー | タップ |

## 置換ルール

1. **完全置換**: プレースホルダーを内容に完全に置き換える
2. **テーブル拡張**: テーブル行のプレースホルダーは複数行に展開可能
3. **セクション追加**: 条件付きセクションは必要時に追加

### 例: テーブル拡張

**Before:**
```markdown
| {{ELEMENT_NAME}} | {{ELEMENT_STATE}} | {{ELEMENT_NOTES}} |
```

**After:**
```markdown
| 送信ボタン | 有効 | 青背景 #0070e0 |
| 送信ボタン | ホバー | 濃い青 #0060c0 |
| 送信ボタン | 無効 | グレー背景 #cccccc |
```
