## AC抽出ルール索引

### spec.md セクション別

| セクション | 抽出対象 | AC種別 |
|-----------|---------|--------|
| UI状態 | 状態遷移条件 | 表示系AC |
| インタラクション | トリガー→アクション | 操作系AC |
| フォーム仕様 | バリデーションルール | 入力系AC |
| 画面フロー | 遷移条件 | ナビゲーション系AC |
| APIマッピング | データバインディング | データ系AC |

### Given-When-Then パターン

```gherkin
# 表示系
Given ユーザーが[画面]を表示している
When [条件]のとき
Then [要素]が[状態]で表示される

# 操作系
Given ユーザーが[画面]を表示している
When ユーザーが[要素]を[操作]する
Then [結果]が発生する

# 入力系
Given ユーザーが[フォーム]に入力している
When [フィールド]に[不正な値]を入力する
Then [エラーメッセージ]が表示される
```

### 優先度マトリクス

| 優先度 | 条件 | 例 |
|--------|------|-----|
| P0 | コア機能、ブロッカー | ログイン、決済 |
| P1 | 主要機能 | 検索、一覧表示 |
| P2 | 補助機能 | フィルター、ソート |
| P3 | エッジケース | エラー処理 |

## 使用方法

1. spec.md の各セクションを解析
2. extraction-rules.md で抽出ルールを確認
3. Given-When-Then 形式でAC生成
4. 優先度を付与して出力
